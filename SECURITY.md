# Політика безпеки

## Повідомлення про вразливості

1. Надішліть приватний лист на security@moment.local з описом проблеми.
2. Очікуйте відповідь протягом 72 годин.
3. Не розголошуйте деталі до випуску виправлення.

## Підтримувані версії

| Версія | Підтримка |
| ------ | --------- |
| main   | ✅        |

## Рекомендації

- Зберігайте секрети лише в `.env` та секретних менеджерах.
- Використовуйте HTTPS у продакшені та обмежуйте доступ до S3/MinIO.
- Регулярно оновлюйте залежності (`pnpm audit`).
